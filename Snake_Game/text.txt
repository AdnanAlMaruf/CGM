Description:

This is classic snake game using OpenGlut,C++. Here are some feature of this game:

1.Speed of Snake will be  increasing as you progress upper level.
2.Easy to learn, Hard to master it.

This game has all features that a classic snake game have.The only difference is that it is created by C++.It will work on all the browsers but work best when you use chrome or safari .If you want to play the you have to press S/s  to start the game. The directional keys are the controller of this game. Happy Snaking……
